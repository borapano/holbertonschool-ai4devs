# Microservices Architecture

- **Auth Service**: Manages user registration, login, authentication, and session validation.
- **Group Service**: Handles creation of groups, member invitations, and management of group participation.
- **Expense Service**: Manages the creation, update, and retrieval of shared expenses within groups.
- **Settlement Service**: Calculates balances and determines how group members should settle debts.
- **Notification Service**: Sends notifications to users when expenses are added or balances change.
- **Analytics Service**: Collects and analyzes expense data to generate insights about spending patterns.
