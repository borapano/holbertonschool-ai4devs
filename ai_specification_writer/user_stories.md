### User Story 1
As a corporate employee, I want to find colleagues with similar commute routes so that I can share rides and reduce my daily transportation costs.

**Acceptance Criteria**:
- User enters home location and work schedule.  
- System suggests colleagues with similar routes within a defined radius.  
- Matching results display compatibility scores.

**Priority**: MVP


### User Story 2
As a corporate employee, I want to create or join a carpool group so that I can commute regularly with the same colleagues.

**Acceptance Criteria**:
- User can create a carpool group with schedule details.  
- Other employees can request to join the group.  
- Group members receive confirmation notifications.

**Priority**: MVP


### User Story 3
As a corporate employee, I want to coordinate schedules with my carpool group so that we can maintain reliable commuting arrangements.

**Acceptance Criteria**:
- Users can view shared schedules within the group.  
- Members can propose schedule changes.  
- Notifications are sent when schedules are updated.

**Priority**: High


### User Story 4
As an employee driver, I want to specify the number of available seats in my car so that passengers know if they can join my ride.

**Acceptance Criteria**:
- Driver can enter vehicle details and seat availability.  
- System limits passengers based on seat capacity.  
- Passengers see available seats before joining.

**Priority**: MVP


### User Story 5
As an employee, I want to receive notifications about schedule changes so that I can adjust my commute plans in time.

**Acceptance Criteria**:
- System sends notifications for schedule changes.  
- Users receive alerts through the app.  
- Notifications include updated time or pickup details.

**Priority**: High


### User Story 6
As an HR manager, I want to see reports on employee participation in carpooling so that I can measure engagement in the program.

**Acceptance Criteria**:
- Dashboard shows number of active users.  
- Reports display weekly and monthly participation rates.  
- Data can be exported for internal reports.

**Priority**: Medium


### User Story 7
As a sustainability officer, I want to track carbon emission reductions from carpooling so that I can report environmental impact.

**Acceptance Criteria**:
- System calculates CO₂ savings per trip.  
- Monthly sustainability reports are generated.  
- Data can be shared with corporate reporting tools.

**Priority**: Medium


### User Story 8
As a facility manager, I want to reserve parking spaces for carpool participants so that we encourage more employees to share rides.

**Acceptance Criteria**:
- Carpool vehicles can register for reserved parking.  
- System tracks active carpool groups.  
- Parking access is granted only to eligible vehicles.

**Priority**: Medium


### User Story 9
As a corporate employee, I want to log in using my company account so that I can access the platform without creating a new login.

**Acceptance Criteria**:
- User can authenticate using corporate SSO.  
- System verifies the employee through the company directory.  
- Access is granted automatically after authentication.

**Priority**: MVP


### User Story 10
As an employee, I want to see my personal cost savings from carpooling so that I understand the financial benefit of using the platform.

**Acceptance Criteria**:
- System calculates estimated fuel savings per trip.  
- Users can view weekly or monthly savings.  
- Savings are displayed in a personal dashboard.

**Priority**: Low
