# Unit Test Generation Prompt Template

**Role**: Software Test Engineer  
**Task**: Generate unit tests for the provided function to verify its correctness.  
**Input Placeholder**: [FUNCTION_CODE]  
**Expected Output**: Set of unit test cases with expected results.

# Test Case Creation Prompt Template

**Role**: QA Engineer  
**Task**: Create test cases to verify that the program works correctly in different scenarios.  
**Input Placeholder**: [PROGRAM_DESCRIPTION] or [CODE_SNIPPET]  
**Expected Output**: List of test cases with inputs and expected outputs.

# Edge Case Detection Prompt Template

**Role**: Software Testing Specialist  
**Task**: Identify edge cases that should be tested to ensure the program handles unusual inputs correctly.  
**Input Placeholder**: [FUNCTION_DESCRIPTION] or [CODE_SNIPPET]  
**Expected Output**: List of edge cases and explanations of why they should be tested.
