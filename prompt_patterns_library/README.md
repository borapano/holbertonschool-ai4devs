# Prompt Patterns Library

## Project Introduction
This project contains a collection of reusable prompt templates designed to support common software development tasks using AI.  
The templates are organized by category and cover typical developer workflows such as improving code quality, debugging issues, generating documentation, and creating tests.  
Each prompt template defines a role, task description, input placeholder, and expected output format to ensure consistent and effective AI interactions.

## Categories Overview
The prompt templates are grouped into four main categories:

- Code Quality – prompts focused on improving readability, structure, and performance of code.
- Debugging – prompts that help identify and fix errors in code.
- Documentation – prompts used to generate clear explanations and technical documentation.
- Testing – prompts designed to generate and improve tests for code.
