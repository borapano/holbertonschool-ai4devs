# Node.js Backend Template with AI Support

## Features
- Pre-configured Copilot settings
- ESLint + Prettier
- AI code review workflow
- AI documentation generation workflow

## Setup
1. Copy `.copilot-settings.yaml` to your project root.
2. Copy `automation/tasks.json` to your automation folder.
3. Run `npm install` for dependencies.
4. Use VS Code Tasks to execute AI workflows.
