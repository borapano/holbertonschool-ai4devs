# Setup Notes – GitHub Copilot Configuration

## IDE
- **IDE Used**: Visual Studio Code  
- **Version**: 1.87.x (or latest stable)

## Extensions Installed
- **GitHub Copilot** (by GitHub)  
- **GitHub Copilot Chat** (optional, for interactive assistance)  
- **ESLint** (for JavaScript linting)  
- **Prettier** (for code formatting)

## Installation Steps

1. Open Visual Studio Code  
2. Go to Extensions Marketplace (`Ctrl + Shift + X`)  
3. Search for **GitHub Copilot**  
4. Click **Install**  
5. (Optional) Install **GitHub Copilot Chat** for conversational AI help  

## Authentication

1. After installation, click **Sign in to GitHub**  
2. Authorize VS Code to access your GitHub account  
3. Ensure your GitHub account has an active Copilot subscription (Free trial or paid)

## Configuration

- Enable Copilot suggestions:
  - Go to **Settings → Search “Copilot”**
  - Ensure **Inline Suggestions** are enabled  

- Optional settings:
  - Enable/disable suggestions per language  
  - Adjust suggestion delay and behavior  

## Usage

- Start typing code → Copilot suggests completions automatically  
- Press `Tab` to accept suggestion  
- Use `Ctrl + Enter` to open Copilot panel with multiple suggestions  
- Use Copilot Chat for explanations, debugging, or generating code snippets  

## Tool Versions

- Visual Studio Code: 1.87.x  
- GitHub Copilot Extension: v1.x.x  
- GitHub Copilot Chat: v0.x.x  

## Notes

- Internet connection is required for Copilot to function  
- Suggestions improve based on context and comments  
- Best results are achieved with clear function names and comments
